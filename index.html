Frequently Asked Questions

What is Bedrock Cosmos? +
Bedrock Cosmos is a custom-built proxy launcher for Minecraft Bedrock Edition that adds custom capes, reveals hidden marketplace packs, and restores missing content like capes, animations, and skins that were removed from official packs.
Is Bedrock Cosmos safe to use? +
Yes! Bedrock Cosmos is completely open source and available on GitHub. You can review the code and build it yourself to see exactly what it does.
Which platforms are supported? +
Currently, Bedrock Cosmos is only available for Windows. It works with any Minecraft Bedrock Edition client installed on Windows systems.
Will this get me banned? +
Bedrock Cosmos operates as a client-side proxy and does not modify game files or give unfair advantages.
Can I upload my own custom capes? +
Yes! We provide templates and tools for you to upload your own. Join our Discord server to submit your capes for use with Bedrock Cosmos.
Does it affect game performance? +
No! Bedrock Cosmos is designed to be lightweight and runs in the background using minimal system resources. You won't notice any performance impact while playing.
How do I get support? +
Join our Discord server where our community and support team are ready to help! You can ask questions, report issues, and connect with other Bedrock Cosmos users.
